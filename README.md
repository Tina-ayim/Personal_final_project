# Community Rental Platform

A  marketplace that allows users to rent out their underutilized household items (like tools, cameras, and camping gear) to neighbors. Built with raw PHP, MySQL, and secure coding practices.

#Key Features

*   Secure Authentication: Login/Register with Rate Limiting and BFCache protection.
*   Dynamic Dashboard: Real-time "Lent vs. Borrowed" analytics using Chart.js.
*   Rental System: Complete cart flow with calculating Service Fees + Insurance.
*   Messaging: Internal secure inbox for renters and owners to communicate.
*   Security First: OWASP-hardened against SQL Injection, CSRF, and XSS.

#Tech Stack

*   Backend: PHP
*   Database: MySQL 
*   Frontend: HTML5, CSS3, JavaScript
*   Environment: XAMPP / Apache
